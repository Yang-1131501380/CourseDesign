# Hardware

该目录用于存放项目公开同步的硬件设计资料，面向仓库展示与工程复现。

## 当前内容

- `CousrseDesign.eprj2`
  嘉立创EDA专业版工程文件，包含本项目硬件设计的核心工程数据。

## 说明

- 当前仓库仅同步项目主硬件设计文件，避免将本地参考资料、第三方开发板配套包和无关附件一并上传。
- 本地仍保留其他参考性硬件资料，但默认不再同步到 GitHub。

## 后续可补充内容

如果后续需要进一步完善硬件部分，建议按下面类型逐步补充：

- 原理图导出 PDF
- PCB 截图或打样图
- BOM 清单
- 硬件模块说明文档
- 接线说明或系统连接框图
