# STM32 无人车实时追踪无人机项目

## 项目简介

本项目面向课程设计场景，完成基于 `STM32F407` 与 `K230` 视觉模组的无人车实时追踪无人机系统设计与实现。系统由 K230 侧完成目标检测与距离信息输出，STM32 侧负责视觉数据解析、二维云台控制、姿态解算与底盘跟随控制，形成完整的“视觉检测 -> 云台跟踪 -> 车体姿态感知 -> 底盘低速跟随”闭环链路。

## 关键硬件

- 主控：`STM32F407`
- 视觉模组：`K230`
- 云台执行机构：`EMM_V5` 二维云台电机
- 姿态传感器：`ICM42688`
- 磁场传感器：`QMC5883`
- 底盘电机驱动：`TB6612`
- 车轮测速：增量式编码器
- 显示模块：`ST7735 LCD`

## 关键软件与算法

- 基于 `USART6` 的 K230 视觉数据接入与协议解析
- 基于 `EMM_V5` 的云台 yaw/pitch 跟踪控制
- 基于 `ICM42688 + QMC5883` 的车体姿态与航向感知
- 基于编码器反馈的底盘双轮速度控制
- 基于目标偏差与距离分段的低速跟随控制策略
- 基于 `RT-Thread` 的控制、监测、IMU 采样、底盘跟随多线程任务架构
- 基于 LCD 与串口日志的运行状态可视化调试
- I2C 通信异常检测与恢复机制

## 控制链路

```text
K230 目标检测
-> USART6 发送目标中心与距离信息
-> STM32 解析目标数据
-> EMM_V5 云台跟踪目标方向
-> ICM42688/QMC5883 提供车体姿态与航向参考
-> 底盘根据目标偏差与距离进行低速跟随
-> LCD/串口输出系统状态
```

## 仓库结构

- `software/`：嵌入式软件工程，包含 STM32 主控代码、底层驱动、控制任务与整车联调代码
- `hardware/`：硬件设计文件，当前包含嘉立创工程文件
